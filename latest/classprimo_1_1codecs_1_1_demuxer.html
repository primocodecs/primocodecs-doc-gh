virtual DemuxerInfo* demuxerInfo ( ) const
pure virtual

Returns a DemuxerInfo object providing information about multiplexed stream.

If the Demuxer input data is provided by a Stream then DemuxerInfo is available after the call to Demuxer::open.

If the Demuxer input data is provided by Demuxer::pushData then DemuxerInfo is available after the first successfull call to Demuxer::getNextSample

Returns
A pointer to DemuxerInfo object. The returned pointer can be NULL.

virtual bool_t getNextSample ( MediaSample *  data,
int32_t  streamNumber = -1 
)
pure virtual

Retrieves the next available sample in the multiplexed stream.

Parameters
data[out] A pointer to MediaSample which receives the next available sample. If the function returns sucessfully MediaSample::streamNumber contains the index of the elementary stream to which the sample belongs.
streamNumber[in] Specifies the stream index for which sample is requested. If the streamNumber parameter is non-negative and the demuxer supports selective demuxing (DemuxFeatureFlags::Selective) the next sample from the specified stream is returned. If the streamNumber parameter is -1 and the demuxer supports sequential demuxing (DemuxFeatureFlags::Sequential) the next available sample from any elementary stream is returned.
Returns
TRUE The demuxer successfully retrieved the next sample, otherwise FALSE.
