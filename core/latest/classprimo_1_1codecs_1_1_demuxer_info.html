Detailed Description

Provides information about the type and properties of the file container, elementary streams and metadata.

Member Function Documentation

virtual Metadata* metadata ( ) const
pure virtual

Returns the demuxer metadata.

Returns
A pointer to a Metadata object. Can be NULL if the input has no metadata.

Normally the caller is not required and should not release the returned object. However it is allowed to retain the Metadata object and it will be valid even when the DemuxerInfo instance is destroyed. In all cases when the caller retains the returned object it should release it when it is not needed anymore.

virtual DemuxerStreamInfoEnum* streams ( ) const
pure virtual

Returns a collection of DemuxerInfo objects describing the audio and video streams (tracks) found in the input.

Returns
A read-only collection of DemuxerInfo objects. Each DemuxerInfo object in the collection represents a stream (track) in the analyzed input.

Normally the caller is not required and should not release the returned object. However it is allowed to retain the collection and it will be valid even when the DemuxerInfo instance is destroyed. In all cases when the caller retains the returned collection it should release it when it is not needed anymore.
