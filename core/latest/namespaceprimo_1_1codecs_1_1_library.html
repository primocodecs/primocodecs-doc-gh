Detailed Description

This namespace contains global functions for the AVBlocks library.

These are C++ wrappers of the functions in the pure "C" interface exported by the library.

The Library namespace serves as class factory and is used to create all PrimoCodecs objects.

Creates a new AAC ADTS demuxer object.

The demuxer supports AAC ADTS format defined in Part 7 of the MPEG-2 standard (known formally as ISO/IEC 13818-7).

primo::codecs::Demuxer* primo::codecs::Library::createMpeg2Demuxer ( )

Creates a new Mpeg2Demuxer object.

The demuxer supports:

MPEG-TS (transport stream)

MPEG-PS (program stream)

MPEG-1 and MPEG-2 Video. Defined in ISO/IEC-11172-2 and ISO/IEC 13818-2.

MPEG-1 and MPEG-2 Audio, Layer I, II and III. Defined in ISO/IEC 11172-3 and ISO/IEC 13818-3.

H.264 (also known as MPEG-4 AVC / Advanced Video Codec). Defined in Part 10 of the MPEG-4 standard (known formally as ISO/IEC 14496-10 and as ITU-T H.264).

The caller is responsible for releasing the object when it is not needed anymore.

int32_t primo::codecs::Library::setLicense ( const char *  license)

Sets a license string.

The supplied license is appended to the library license state.

Parameters
licenseA string with license information. If the license string is NULL the library license state is cleared.
Returns
The status after the license is set. The returned value is a combination of flags from the LicenseStatusFlags enum.
Remarks
A Demo build of the library cannot be licensed and always returns LicenseStatusFlags::DemoBuild.
See Also
primo::license::LicenseStatusFlags::Enum
