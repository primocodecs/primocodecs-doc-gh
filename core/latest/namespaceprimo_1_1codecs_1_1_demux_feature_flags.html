Detailed Description

Describes the demuxer capabilities / features.

In practical terms these flags specify which parts of the full Demuxer interface can be used with a specific Demuxer implementation.

Enumeration Type Documentation

enum Enum

DemuxFeatureFlags.

Enumerator
None 

The demuxer features are not set or not known.

PullSequential 

The demuxer pulls data from an input stream.

The output samples are returned as they come in the input stream. By default the demuxer can return samples without seeking in the input stream. If an implemetation needs to seek in the input stream in order to return samples it must add the PullSequentialNeedsSeekableStream flag in addition to the PullSequential flag.

PullSelective 

The demuxer pulls data from an input stream.

The output samples are returned from the selected elementary stream. In this mode the demuxer is assumed to seek in the input stream, so it must be seekable.

Push 

Input data is pushed to the demuxer.

The output samples are returned sequentially as they come in the input.

CanSeekTime 

Can seek to the specified time (Demuxer::seekTime) as long as the input stream is seekable.

PullSequentialNeedsSeekableStream 

A demuxer in PullSequential mode needs a seekable input stream in order to parse the input and to return samples from it.
